# AI Evaluation Studio — 视觉设计规范

> 版本：v0.1 · 2026-05-25
> 风格定位：暗色 SaaS 专业工具感（Klein + Linear 谱系）
> 关键词：数据感、克制、精致、科技感但不花哨

---

## 1. 整体氛围

- **暗色为唯一主题**（MVP 不做亮色切换，精力放在暗色打磨上）
- 专业但不冰冷：有色彩呼吸感，不是纯黑纯灰
- 数据可视化是视觉高潮，交互页面是沉稳工具
- 类似 Linear 的"少即是多"：留白 > 装饰

## 2. 配色系统

### 2.1 基础色板

```
背景层级（从深到浅）：
  ┌─ bg-base:       #09090b    最深底色，body背景
  ├─ bg-card:       #141419    卡片/面板背景
  ├─ bg-elevated:   #1c1c24    悬浮层/弹窗背景
  ├─ bg-hover:      #25252f    hover 态背景
  └─ bg-active:     #2e2e3a    active/选中态背景

边框层级：
  ├─ border-subtle: #1e1e2a    卡片分隔线
  ├─ border-default:#2a2a3a    默认边框
  └─ border-strong: #3a3a4f    强调边框（输入框 focus 等）

文字层级：
  ├─ text-primary:  #f0f0f5    标题/正文
  ├─ text-secondary:#a0a0b8    次要信息/描述
  ├─ text-tertiary: #6b6b80    占位符/禁用态
  └─ text-inverse:  #09090b    反色文字（用于深色背景上的文字）
```

### 2.2 强调色

```
主强调色 — 紫蓝渐变（品牌色）：
  ● primary:       #7c5cfc    主按钮/链接/选中态
  ● primary-hover: #9070ff    
  ● primary-muted: rgba(124,92,252,0.15)  浅底标签/背景高亮

渐变（用于品牌位：Logo/大标题/报告高光）：
  gradient: linear-gradient(135deg, #7c5cfc 0%, #3b82f6 100%)

功能色：
  ● success: #10b981    得分上升/通过/完成
  ● warning: #f59e0b    得分下降/需注意
  ● danger:  #ef4444    失败/低分
  ● info:    #3b82f6    提示/信息

图表配色（多系列对比，确保色盲友好）：
  series-1: #7c5cfc   紫蓝（组合A）
  series-2: #3b82f6   蓝   （组合B）
  series-3: #10b981   翠绿（组合C）
  series-4: #f59e0b   琥珀（组合D）
  series-5: #ef4444   红   （组合E，最多5组对比）
```

### 2.3 特效色

```
毛玻璃卡片：
  background: rgba(20, 20, 25, 0.6)
  backdrop-filter: blur(12px)
  border: 1px solid rgba(255,255,255,0.06)

光晕（用于报告页头部装饰）：
  box-shadow: 0 0 80px 20px rgba(124,92,252,0.08)

渐变边框（用于重要卡片）：
  border-image: linear-gradient(135deg, #7c5cfc33, #3b82f633) 1
```

## 3. 排版

### 3.1 字体

```
主字体：Inter（西文）+ 系统中文字体
  font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

代码/数据：
  font-family: 'JetBrains Mono', 'Fira Code', monospace;

字号体系（基于 Tailwind 默认，微调）：
  text-xs:   12px  — 标签、辅助说明
  text-sm:   14px  — 表格内容、描述
  text-base: 16px  — 正文、表单
  text-lg:   18px  — 小标题
  text-xl:   20px  — 卡片标题
  text-2xl:  24px  — 页面标题
  text-3xl:  30px  — 数据大数字（得分 4.2/5）
  text-5xl:  48px  — 报告页核心得分
```

### 3.2 字重

```
font-normal:  400  — 正文
font-medium:  500  — 标签、小标题
font-semibold:600  — 卡片标题
font-bold:    700  — 页面标题、数据数字
```

## 6. 图表风格

### 6.1 雷达图

```
背景网格：border-default (#2a2a3a) 虚线
填充：series 色半透明 (opacity 0.15)
边线：series 色 实线 (opacity 0.8)
数据点：series 色 实心圆 r=4
尺寸：300×300
动画：入场 500ms ease-out
```

### 6.2 折线图（趋势图）

```
网格：border-subtle
坐标轴：text-tertiary, text-xs
线条：series 色 stroke-width=2
数据点：series 色 实心圆 r=3，hover r=5
面积：series 色渐变 (opacity 0.08 → 0)
动画：入场 800ms ease-out
```

### 6.3 得分大数字

```
综合得分展示：
  font: text-5xl, font-bold, font-numeric
  颜色：
    ≥ 4.0: success (#10b981)
    3.0-3.9: primary (#7c5cfc)
    < 3.0: warning (#f59e0b)
  
  后缀 "/5"：text-2xl, text-secondary
  
  变化值 (↑0.3 / ↓0.2)：
    上升：success, text-sm
    下降：danger, text-sm
```

## 11. 设计原则总结

1. **暗色一镜到底** — 不做亮色切换，把暗色做到极致
2. **数据是主角** — 得分数字大且醒目，图表干净不装饰
3. **紫蓝品牌色贯穿** — 从 Logo 到按钮到图表到光晕，统一识别
4. **克制的毛玻璃** — 只在报告页 hero + 弹窗用，不过度
5. **图表色盲友好** — 5 色方案通过色盲模拟验证
6. **Klein 级精致** — 1px 边框、4px 间距对齐、hover 微动效
